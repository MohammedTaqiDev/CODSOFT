## Tasks

### Task 1 - Rule-Based Chatbot

**Description:**

* Developed a simple rule-based chatbot.
* Responds to user queries using predefined rules and `if-else` statements.
* Demonstrates basic Natural Language Processing concepts and conversation flow.

**Technology Used:**

* Python

---

### Task 2 - Tic-Tac-Toe AI

**Description:**

* Implemented a Tic-Tac-Toe game where a user plays against an AI.
* The AI uses the **Minimax Algorithm** to make optimal moves.
* The AI is designed to be unbeatable.

**Technology Used:**

* Python
* Minimax Algorithm

---

### Task 3 - Recommendation System

**Description:**

* Built a simple content-based recommendation system.
* Recommends movies based on the user's preferred category.
* Demonstrates the fundamentals of recommendation systems.

**Technology Used:**

* Python
